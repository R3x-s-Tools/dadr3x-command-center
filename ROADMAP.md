# Dad_R3x Command Center Pro Roadmap

Dad_R3x Command Center Pro is an AI-powered stream producer dashboard for OBS, Twitch, viewer analytics, and creator coaching.

## Current Focus

### v0.2.1 - Release Engineering

Goal: make the project easier to build, release, download, and verify.

Status: In progress

Planned completion criteria:

- [ ] GitHub release workflow builds source ZIP
- [ ] GitHub release workflow builds Windows ZIP
- [ ] GitHub release workflow builds macOS Apple Silicon ZIP
- [ ] GitHub release workflow builds macOS Intel ZIP
- [ ] SHA256 checksums generated
- [ ] Local macOS build tested
- [ ] Source ZIP tested
- [ ] Changelog updated
- [ ] Version file updated
- [ ] Release checklist completed

## Upcoming Milestones

### v0.3.0 - Mission Control UI

Goal: redesign the dashboard into a polished, glanceable command center.

Planned features:

- Modern dashboard layout
- Stream health cards
- Live viewer graph
- Chat activity graph
- Event timeline
- Better status colors
- Cleaner AI Producer panel
- Improved analytics display
- Better empty states
- Better first-run experience

### v0.4.0 - Event-Driven AI Producer

Goal: make the AI respond to meaningful stream events instead of mostly timed refreshes.

Planned features:

- AI triggers for new chatters
- AI triggers for returning viewers
- AI triggers for viewer spikes
- AI triggers for follows, subs, bits, raids
- AI triggers for OBS issues
- AI cooldown and priority system
- Context-aware live coaching
- Better clip suggestions

### v0.5.0 - Viewer Intelligence

Goal: help the streamer recognize and build relationships with viewers.

Planned features:

- Viewer profile cards
- Returning viewer detection
- Regular viewer milestones
- Topic memory
- Engagement scoring
- First-time chatter prompts
- Community growth insights

### v0.6.0 - Advanced Analytics

Goal: turn stream logs into useful long-term insights.

Planned features:

- Stream comparisons
- Game performance comparisons
- Best time/day analysis
- Viewer retention trends
- Chat engagement trends
- Clip potential trends
- Post-stream score history

### v1.0.0 - Public Release

Goal: make the application stable and polished enough for other creators to download and use.

Planned features:

- Signed builds
- Installer packages
- Better onboarding
- User-friendly settings editor
- Documentation site
- Stable plugin architecture
- Public release notes
