# App Icon Setup

The PyInstaller spec is now ready for icons.

## Expected files

```text
assets/icon.icns
assets/icon.ico
assets/icon.png
```

## macOS

Uses:

```text
assets/icon.icns
```

## Windows

Uses:

```text
assets/icon.ico
```

## Notes

Do not commit random generated icon experiments. Only commit final icons.

If no icon file exists, the app still builds with the default icon.
